Element types

  • Block Elements are structural or container elements
    • html, body, nav, section, article, headings, paragraphs, lists, forms, footers, etc.
    • Block elements are nested to structure a document
    • The minimum block elements to structure a document?

  • "Self-closing" (or empty elements) have only attributes defining the content and are closed by />.
    <img src="fun.png" alt="kids playing" />

  • With errors, also "quirk mode"
  • XHTML is XML well-formed and valid to a DTD

    • Use W3C validator for conformance to a DTD

Rules & Specificity
  • simple selectors: element, class, id 
    <p id="special" class="general">
   some content
</p>
  • Cascading and Specificity, which declaration wins? 
    p { color: red; }
#special { color: green; }
.general { color: blue; }
    • last declration with the highest specifity wins.
    • class has higher specificity than element type p
    • id has higher specificity than class
    • Which colour wins?
    • developer tools: Styles and Computed panels
  • Pseudo-classes 
    • /* These styles will style our link in all states */
a {
   color: blue;
   font-weight: bold;
}

/* declare visited links to be the same
   color as non visited links */
a:visited {
   color: blue;
}

/* Highlight the link when it is hovered over (mouse over),
   activated (mouse down) or focused (keyboard) */
a:hover,
a:active,
a:focus {
  color: darkred;
  text-decoration: none;
}
  • At-rules — convey metadata 
    • /* import another CSS file */
@import "layout.css";
@media (min-width: 801px) {
  body {
    margin: 0 auto;
    width: 800px;
  }
}
Layout & Box model

The CSS box model is the foundation of layout on the Web

Types of CSS boxes (set with disply property)

  • block — stacked like paragraphs with padding, border, margin
  • inline — opposite to Block; flows with parent block
  • inline-block — between block & inline! can be sized & move with the parent block

For children of the box and complex layout:

  • display: flex
  • display: grid

Temporal data (Audio, Video)

  • Remember MIME types
    • MIME (Multipart Mail Extensions) or Media Type is a standard that indicates the nature and format of content.
      Example MIME types
      • text/html, text/css, text/javascript
      • image/jpeg, image/png, image/svg+xml
      • audio/mpeg, audio/vorbis
      • font/woff, font/ttf
      • video/mp4, video/ogg
      • multipart/form-data
    • Developer tools, Network
      (note the MIME type of each request)
  • It's all in bits! Text, Image, Audio, Video
    • Text ‐ unicode
    • Image ‐ Lossy vs Lossless, Vector vs Raster
    • Audio and Video
      • Analog to Digital conversion (A/D)
      • Audio quality: Bandwidth, Sample rate
        (8kHz human speech, 96kHz high resolution audio)
      • Container formats: WebM, MP4, Ogg
