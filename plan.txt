1. Ignore the preambles and general explanatory notes.
2. There is a Content page identifying each lesson, generate separate sections for each entry in the content page.
3. Vocabulary and standalone words become model "word" type.

5. Structure in the PDF becomes Grammer Notes in tha App.
5. Activity for the JSON output for both document and section level data is:

        "sentences": true
    },
